# Study Planner App
A web-based application built using Next.js to help students organize tasks and improve productivity.

 Routes
- `/` → Home page
- `/tasks` → Task management page
- `/about` → About page

 Components
- **Navbar**: Handles navigation between pages  
- **TaskForm**: Allows users to add tasks  
- **TaskList**: Displays all tasks  
- **TaskCard**: Represents a single task  

 State Management
- Managed using React `useState` in the Tasks page  
- Handles:
- Adding tasks  
- Marking tasks as complete  
- Deleting tasks  

 Features
- Add tasks  
- Mark tasks as complete  
- Delete tasks  
- Responsive design  
- Navigation between pages  

 Technologies Used
- Next.js
